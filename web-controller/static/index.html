iOS Setup
  1. On iPad: Settings → MIDI Network Session
  2. Under Directory, tap YeeSiteLights → Connect
  3. Open GarageBand — MPK Mini notes now control lights
  4. If not found: tap + and enter this server's IP, port 5004
